# House Price Prediction

## Overview
This project predicts house prices using the Ames Housing Dataset.

## Models Used
- Linear Regression
- Ridge Regression
- Random Forest Regression

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Results
Random Forest achieved the best performance with an R² score of approximately 0.876.

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
